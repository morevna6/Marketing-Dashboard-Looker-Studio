# Marketing Dashboard (Looker Studio)

## Project Overview
This project presents an interactive marketing dashboard built using Looker Studio.

The dashboard is designed to analyze campaign performance and key advertising metrics.

---

## Dashboard Features

### Key Metrics
- Ad Spend  
- CPC (Cost Per Click)  
- CPM (Cost Per 1000 Impressions)  
- CTR (Click-Through Rate)  
- ROMI (Return on Marketing Investment)  

---

## Visualizations
- Time series analysis of Ad Spend and ROMI  
- Campaign performance comparison  
- KPI table with conditional formatting
